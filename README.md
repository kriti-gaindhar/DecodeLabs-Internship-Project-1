# 🌐 Personal Portfolio Website

### ✨ A Modern Responsive Portfolio Website
Designed and developed using **HTML5** and **Tailwind CSS** with a focus on responsiveness, clean UI, accessibility, and modern web design.

## 📖 About The Project
This project is a modern multi-page personal portfolio website developed during the **DecodeLabs Internship**.
The website showcases professional information, technical skills, services, and contact details through an elegant and responsive user interface. The design follows modern UI principles while ensuring compatibility across desktop, tablet, and mobile devices.

---

## ✨ Features

- 🏠 Modern Landing Page
- 👩 Professional About Section
- 💼 Services Showcase
- ⚙ Technical Specifications Page
- 📞 Contact Form
- 📱 Fully Responsive Design
- 🎨 Tailwind CSS Components
- 🚀 Clean and Semantic HTML5
- 📌 Sticky Navigation Bar
- 🌟 Smooth Hover Animations
- ♿ Accessibility Friendly
